### 1.安装sbt

1. windows 平台
    * win10系统
    * 通过msi文件安装 安装完成后发现path下多了一条
      * D:\develop\tools\generators\sbt\bin
      * sbt是安装在D:\develop\tools\generators\sbt
      * 看来像java一样设置一下环境变量便可以了
    * 通过zip文件安装
      * 添加一个换件变量名为SBT_HOME
      * 向path中添加%SBT_HOME%\bin
